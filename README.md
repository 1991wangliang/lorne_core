
### 框架介绍：

  项目框架底层基于spring研发，项目框架被设计的初衷，主要是两方面：  
  1. 服务于其他架构体系
  2. 针对一些第三发框架的第二次优化封装。



### 框架支持功能如下：

1. 基于springmvc封装了RestController   
2. 定义了db框架的注解  
3. 定义了项目的异常  
4. 整合了Redis框架的RedisUtils框架  
5. 整合了apache http框架  
6. 封装了AES Base64 Md5加密算法处理类  
7. 封装了Task线程等待唤醒机制  
8. 封装了Thread本地多线程同步数据汇总框架  
9. 封装了本地配置资源文件读写    
10. 封装了DateUtils MathUtils KidUtils JsonUtils 常用工具类封装
